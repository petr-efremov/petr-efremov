# 👋 Привет! Я Пётр Ефремов | Senior Python Backend Developer

**7+ лет в коммерческой разработке**

Специализируюсь на отказоустойчивых распределенных системах (Highload, микросервисы) и надёжной бизнес-логике (**Reliability-First Design**). 

Создаю масштабируемые API, устойчивые к отказам внешней среды. Фокусируюсь на чистоте кода (SOLID, DDD), строгой консистентности данных в распределенной среде и сквозной прозрачности систем изнутри (Distributed Tracing).

Ищу позицию в компании со зрелой инженерной культурой для решения сложных продуктовых и технических задач.
